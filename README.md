# Proyecto Final
Sistema de Gestión de Ventas e Inventario para Tienda
El progtrama está separado en varios archivos ya que facilita la organización del código.
main.cpp se encarga del menú principal, producto.cpp administra los productos, venta.cpp
controla las ventas y reportes.cpp genera los reportes. Esta estructura hace que el programa
sea más fácil de mantener, corregir y ampliar, además de cumplir con el principio de 
modularización solicitado en el proyecto. Espero que sea de su agrado.
